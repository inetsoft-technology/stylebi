This example uses the full page width so the node spacing and line routing read like a complete worksheet graph instead of a cramped card sketch.
Selected node CSS: .worksheet-node.selected uses a soft blue header #DDF1F5, a cool border #BFDDE5, and a restrained halo so the active block leads the eye without becoming heavy.
Connected node CSS: .worksheet-node.connected .worksheet-node-header applies the pale yellow header #FEF8EE while the card body stays white, keeping related steps visible but clearly secondary.
Primary node CSS: .worksheet-node.primary .worksheet-node-icon paints only the icon tile in amber #EEA555; the rest of the node stays quiet so the orange signal reads as a role marker rather than a full alert block.
Line and dimming CSS: .worksheet-line-active uses dark charcoal #242421, .worksheet-line-muted uses #B8B3AA, and .worksheet-node.dimmed lowers opacity so unrelated paths recede.
